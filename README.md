# E-Commerce Web Application 🛒

## Description

A Django-based E-Commerce web application that allows users to browse products, add items to cart, register/login, and place orders.

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite
* JavaScript

## Features

* User Registration and Login
* Product Listing
* Product Details Page
* Shopping Cart
* Order Management
* Stripe Payment Integration

## How to Run

```bash
pip install -r requirements
python manage.py runserver
```
